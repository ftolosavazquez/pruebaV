Process Document from url and extract all the fields from it.

Parameters
string | null$file_urlRequired if file_urls isn't specified. Publicly accessible URL to a file, e.g. "https://cdn.example.com/receipt.jpg".
array | null$file_urlsRequired if file_url isn't specifies. List of publicly accessible URLs to multiple files, e.g. ['https://cdn.example.com/receipt1.jpg', 'https://cdn.example.com/receipt2.jpg']
array | null$categoriesArray of categories to use when categorizing the document
bool$delete_after_processingDelete this/these document(s) from Veryfi after data has been extracted
int$boost_modeFlag that tells Veryfi whether boost mode should be enabled. When set to 1, Veryfi will skip data enrichment steps, but will process the document faster. Default value for this flag is 0.
string | null$external_idOptional custom document identifier. Use this if you would like to assign your own ID to documents.
int | null$max_pages_to_processWhen sending a long document to Veryfi for processing, this parameter controls how many pages of the document will be read and processed, starting from page 1.
array$additional_request_parametersAdditional request parameters.
Returns
string Data extracted from the document.
